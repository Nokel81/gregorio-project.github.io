Adding annotations

All these details of the output are controlled by the TeX file which represents the document, so let's look at that.

kyrie-17-13b-main.tex
\documentclass[12pt, a5paper]{article}
\usepackage{fullpage}
\usepackage{newcent}
\usepackage[T1]{fontenc}
\usepackage[autocompile]{gregoriotex}
\pagestyle{empty}
\begin{document}


\begin{center}\begin{huge}\textsc{Kyrie XVII}\end{huge}\end{center}

\grechangedim{afterinitialshift}{3.2mm}{scalable}
\grechangedim{beforeinitialshift}{3.2mm}{scalable}

\grechangestyle{initial}{\fontsize{43}{43}\selectfont}

\greannotation{\small \textsc{\textbf{}}}
\grecommentary{{\small \emph{}}}
\gregorioscore{kyrie-17-13b}

\end{document}

The \grecommentary{} macro generates the small text above the first stave, so let's add the desired text to it:

\grecommentary{{\small \emph{XIV. s.}}}

The macro \greannotation{} generates the mode label before the first clef, so let's make that:

\greannotaion{\small \textsc{\textbf{VI.}}}

Changing the typeface

To see how the score looks in Palatino font, I looked for the line that sets the typeface New Century Schoolbook:

\usepackage{newcent}

and I changed it:

\usepackage{palatino}

Here's how the score looks after that change:

The best-looking free font for gregorio is a matter of taste, but kpfonts is also a good choice (\usepackage{kpfonts}).

Changing vertical spacing

To add more space between the staves, I adjusted a spacing variable by adding the following line:

\grechangedim{spaceabovelines}{1.45576 cm plus 0.36461 cm minus 0.09114 cm}{scalable}%
