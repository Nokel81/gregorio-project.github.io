Contact

You can contact the members of the Gregorio project by several means:

Mailing lists

Two mailing lists are available:

  • gregorio-users@gna.org, the users' list. Here you can ask your questions about gregorio usage and get help. There is also a searchable archive of past discussions on this list.
  • gregorio-devel@gna.org, the developers' list. You can use this list to report bugs if you don't have a GitHub account (see below). This is also the list to use if you're interested in getting involved in development.

In order to subscribe to those lists, please see the mailing-lists page on gna.org.

The user mailing list is also accesible as a GMane news feed.

IRC

An IRC channel is available on Freenode, with the channel name #gregorio.

Bugs

Bug reporting is done through GitHub (see the Guidelines).

Website

This website is hosted using GitHub Pages. You can find its repository here, and can report typos, innacurate or unclear information, and other problems here.
