Contact

You can contact the members of the Gregorio project by several means:

Mailing lists

Three mailing lists are available:

  • gregorio-users@gna.org, the users' list. Here you can ask your questions about gregorio usage and get help. There is also a searchable archive of past discussions on this list.
  • gregorio-devel@gna.org, the developers' list

In order to subscribe to those lists, please see the mailing-lists page on gna.org.

IRC

An IRC channel is available on Freenode, with the channel name #gregorio.

Bugs

Bug reporting is done through Github (see the Guidelines).

The code of this website is hosted at Github, bugs can be reported on the bugtracker.
