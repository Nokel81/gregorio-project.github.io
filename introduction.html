Introduction to Gregorio

So, you've installed TeXLive (or some other TeX distribution) and Gregorio and are now ready to get to work. But how do you do that? If you're familiar with TeX, you probably have some ideas already as to the workflow, but if this is your first experience with TeX then you're probably stuck looking for what program to open up to get started.

The fisth thing to understand is that TeX and all the flavors built on top of it (LaTeX, LuaTex, LuaLaTeX, etc.) are basically compilers. So too is Gregorio. This means that they are editor agnostic. You can write the source that gets fed to these programs in any editor with which you feel comfortable so long as the editor can save the file as plain text (i.e. no formatting). It's a very similar workflow to programming. In this arangement Gregorio functions as a pre-compiler. It translates a source document writen in gabc (a standard designed to make entering Gregorian Chant fairly user friendly) into something that can be placed in a LuaTeX (or LuaLaTeX) document using the gregoriotex package.

What editor you use is largely up to you, but if you've never worked with TeX before then I highly recommend using an editor specifically designed to work with TeX. Luckly TeXLive comes with just such an editor: TeXworks. Just follow the link below that corresponds to your system to see how to get started.
