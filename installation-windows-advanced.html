This page is made for advanced Windows users who want to have more control over gregorio, or who want the latest development version.

Compiling gregorio under Windows is unfortunately rather lengthy, as it requires the installation of a Cygwin system and TeXLive, if you do not have any of these already installed.

Installing Cygwin

The first step to get gregorio working under Windows is to install Cygwin. Cygwin is an Linux abstraction layer for Windows which contains many useful programs, notably a command-line interface.

To install Cygwin, download the installer on the official site and launch it, take all the default options, and choose a mirror site: any of them will suffice, in case you're not familiar with what a mirror site is. You will arrive at a window for choosing which software you want to install within your cygwin system. You will need the following supplementary software packages: flex, bison, gcc, cvs, gcc4, make, automake, autoconf, libtool, python, libxml2, libxml2-dev, gettext, gettext-dev, pkg-config.

/!\ Warning!!! Do not install tetex, tetex-base, tetex-bin, tetex-extra nor TeXMacs, or you won't be able to use TeXLive, which is mandatory for gregorio!

Once you have made your choice of software, go on to the installation proper. Downloading the packages may take a little time. When that is done, you can launch Cygwin.exe, which will open a command-line terminal. To start a more convenient graphical interface, just type startx in that terminal.

You can use a command-line terminal to familiarise yourself with the Linux shell commands. If this is your first such experience, refer to the numerous tutorials on this subject which are on the net. You can find an appropriate tutorial by searching with Google, with the keywords tutorial shell linux.

Installing TeXLive

To use GregorioTeX, the TeX system of gregorio, you will need a complete TeX system (called a distribution). There are various TeX distributions for Windows. The most common are:

  • MikTeX, a distribution exclusively for Windows,
  • TeTeX, a multi-platform distribution containing Aleph but not LuaTeX, and no longer maintained,
  • TeXLive, an up-to-date widespread multi-platform distribution,

As TeXLive is the system used on Linux and Mac, we have chosen to describe the installation of TeXLive. Everything should work the same with MikTeX but it has not been tested.

To begin, download the latest TeXLive installer from the official site and unzip it. Then you have two options:

The first is to launch install-tl.bat and to say yes all the time, it will simply install all the TeXLive distribution (about 3GB).

If you have less time or disk space, launch install-tl-advanced.bat. A graphical interface will open to give you the installation options. Click on the uppermost "Change" button, to select the scheme medium. Click then on the second "Change" button and select at least "LuaTeX packets", "LaTeX3 packets", "LaTeX recommended packages", "Recommended standard packages", "Recommended fonts", "LaTeX base packages" and "Essential programs and files". Click then on the "Change" button for languages. Deselect all the documentation languages (you can find them on the Internet), as well as all the languages on the left that you won't use (don't forget to check latin, which is unchecked by default). To gain some room, you can also say "no" to "Install source of macros and fonts". It should be less than 1GB.

Once you have activated the installation choices, click on "Install TeXLive". The installation may take a while, because it requires downloading the necessary packages.

Compiling Gregorio

Compiling Gregorio is very easy under windows, if you have the good dependencies (see above on this page), you just have to compile like on any other civilized system. First download the source tarball from the download page or clone the git reposity, and save it in your Cygwin home directory, typically C:\\cygwin\home\foo, where foo is your Windows username.

Then launch Cygwin and, if you have downloaded the sources, uncompress the gregorio archive:

tar xzf gregorio-*.tar.gz

Enter the gregorio directory with the command:

cd gregorio-*

And finally compile and install with the commands:

$ ./configure --prefix=/usr
$ make
$ make install
$ cd fonts
$ make install
$ texhash
