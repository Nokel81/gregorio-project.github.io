Configuring TeXShop for Gregorio

Automatic score compilation

Gregorio needs a lualatex that is able to launch the gregorio program during compilation, in order to compile gabc scores automatically.

TeXworks can be configured to do so by creating a new engine for compilation that adds the --shell-escape argument to lualatex.

To create it, open the folder ~/Library/TeXShop/Engines in Finder. Copy the LuaLaTeX engine from the Inactive/luaTeX folder, and modify it by opening it with text editor (such as Textedit) and adding "--shell-escape" right after "lualatex". Save this file as ~/Library/TeXShop/Engines/LuaLaTeX+se. This enables GregorioTeX to automatically compile your gabc scores as needed when LuaLaTeX compiles your document.

The name LuaLaTeX+se is arbitrary, but it's important to use it, so that everyone can have the same scores.

Ease of use of .gabc files (optional)

If you want to be able to filter files with .gabc extensions in the file dialogs of TeXShop, open a Terminal window (it's in Applications/Utilities) and enter the following command:

defaults write TeXShop OtherTeXExtensions -array-add "gabc"
