The Gregorio project

The Gregorio project offers tools for typesetting Gregorian chant. These tools include:

  • gabc: a notation for representing Gregorian chant using ASCII characters
  • GregorioTeX: a TeX style for typesetting scores
  • a software application to convert from gabc to GregorioTeX

Together, these tools, added to a TeX installation, allow the user to engrave beautiful Gregorian chant scores.

The Gregorio project consists of 100% free software.

This website contains an introduction, documentation and tutorial for Gregorio versions 4.0 and later. For information on version 3.0, you will need to access the history in the GitHub repository for this website. For information on older versions (2.4.2 and earlier), please see the old website.
