Installing Gregorio

If you have a full installation of TeXLive 2016 (including MacTeX), then Gregorio 4.1 is already installed on your system as part of that installation. If you use a different TeX distribution, have an older version of TeXLive, or want a newer version of Gregorio, then refer to the following pages:

  • installation under Windows
  • installation under GNU/Linux
  • installation under MacOS X

Editor Configuration

If you plan to use an editor, then you'll need to configure it once Gregorio is installed. For instructions on how to do this, refer to the following pages:

  • TeXworks (a cross platform editor for Linux, Mac, and Windows)
  • TeXShop (a Mac only editor)
